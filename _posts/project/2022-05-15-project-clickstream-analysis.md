# 프로젝트 목표

- 다양한 곳에서 웹서비스 형태로 제공되는 채용관련 정보를 데이터로 모아 검색/시각화 형태로 서비스를 제공하고자 한다
- 이 프로젝트는 크게 두 단계로 계획하고 있다
    - **1단계**: 개발자 모드 - 코드 실행으로 검색/시각화 구현
    - **2단계**: 데모 모드 - 간단한 프론트엔드를 통해 검색/시각화 서비스 제공
