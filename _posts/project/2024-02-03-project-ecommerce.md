# 기획 배경

- 웹서비스의 대표적인 분야인 이커머스를 주제로 프로젝트를 만들었다
- 사용자 검색어 기반으로 스마트한 검색 결과를 제공한다 (ex. 바지 기장이 100cm인 검은색 긴 바지)
- 옵션 선택에 따라 다른 이미지를 제공한다
- 후불 결제

# 개발 환경

- **프론트엔드(FE)**
  - Next.js
- **백엔드(BE)**: 
  - NestJS
- **CI/CD**: 
  - Docker
  - AWS ECR
  - AWS EKS
  - Github
  - Github Action
